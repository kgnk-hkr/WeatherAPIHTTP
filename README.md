## 使用環境
jqがインストールされていること

## batファイルの実行手順
[httptest.bat]をダブルクリックします。  
コマンドプロンプトが起動します。  

お天気を知りたい場所の郵便番号を入力します。
デフォルトは100-0005です。
デフォルトを使用する場合は何も入力せずEnterキーを入力します。

事前にOpenWeatherで取得したAPIキーを入力します。
https://openweathermap.org/
  
任意のキーを押して、コマンドプロンプトを終了します。  
  
[httptest.bat]と同じフォルダ配下に下記のファイルが生成されます。  
	日付_GET実行結果.txt
